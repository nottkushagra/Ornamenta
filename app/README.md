# Ornamenta — App

This is the Ornamenta jewellery web application built with [Next.js](https://nextjs.org), TypeScript, and TailwindCSS.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Pages

- `/` — Homepage with hero section, featured products, craft philosophy, and brand statement
- `/collection` — Full product collection with search, category filtering, and price sorting

## Tech Stack

- [Next.js 16](https://nextjs.org) — React framework with App Router
- [TypeScript](https://www.typescriptlang.org/) — Type safety
- [TailwindCSS v4](https://tailwindcss.com/) — Utility-first CSS

## Deploy on Vercel

The easiest way to deploy this app is to use the [Vercel Platform](https://vercel.com/new).

Check out [Next.js deployment documentation](https://nextjs.org/docs/app/building-your-application/deploying) for more details.
